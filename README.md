# CSOL2_Launcher
This launcher is used to start Counter-Strike: Online 2 (CSO2 CSOL2)

启动器界面仿国服端启动器 深度定制UI 兼容所有用OpenVPN搭建的CSOL2服务器

### 额外功能 - 启动器允许你添加命令行参数激活额外功能

最低画质 -performance 高画质 -quality

分辨率

-720 -1080 -2k -4k

以上画质或分辨率参数任选一个即可

服务器选项

默认 -fc
晴雪 -qx
本地 -local
